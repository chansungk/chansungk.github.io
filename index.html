1. 연구목적¶

  • 기술적 분석 지표와 머신러닝 기법을 이용하여 주가의 단기 방향성을 예측하고자 함

2. 연구배경 및 선행연구¶

  • 주식의 시가, 저가, 고가, 종가, 거래량을 기초로 추세, 패턴 분석 등을 통해 주가를 예측하는 기술적 분석 지표는 주가 예측 연구에서 많이 쓰이는 지표(Bustos et al., 2020)
  • 기술적 분석 지표를 데이터로 주가 예측 시 랜덤포레스트의 성능이 SVM, 인공신경망, 나이브 베이즈 보다 우수(Patel et al., 2014)
  • 뉴스 데이터의 감성 분석과 기술적 분석 지표를 이용, SVM 모델링을 통해 주가 예측 시도(Ren et al., 2018)
  • 강화학습 앙상블 기법을 이용한 데이트레이딩 전략 연구(Carta et al., 2020)
  • 캔들 차트 이미지 학습을 통한 코스닥 등락 예측(송현정 등, 2018)
  • 딥러닝 분석과 기술적 분석 지표를 이용한 코스피 지수 방향성 예측(이우식, 2017)

  • 모델 학습을 위해 OLHCV를 토대로 생성된 아래의 기술적 분석 지표를 변수로 사용

Given :

  • $L_t, H_t, C_t, V_t = t시점의\,저가,\, 고가,\, 종가,\, 거래량$
  • $n = 윈도우 \,사이즈$

    종가 변화율 : $\Delta C = (C_t - C_{t-1})\, /\, C_t-1$

    거래량 변화율 : $\Delta V = (V_t - V_{t-1})\, /\, V_t-1$

    MFM : $\frac{(C_t - L_t)-(H_t - C_t)}{H_t - L_t} \times V_t$

    MOMENTUM : $C_t - C_{t-(n-1)}$

    SMA : $\frac{C_t + C_{t-1}+...+C_{t-(n-1)}}{n}$

    WMA : $\frac{nC_t + (n-1)C_{t-1}+...+C_{t-(n-1)}}{n+(n-1)+...+1}$

    이격률 : $\frac{C_t}{SMA} \times 100$

# 윈도우 사이즈에 따른 데이터셋 추출 함수
def make_dataset(data, label, window_size):
    x_list = []
    y_list = []
    for i in range(len(data) - window_size):
        x_list.append(np.array((data.iloc[(i+1):(i+1)+window_size])).flatten().tolist()) # 2차원 배열로 축소
    
        # 주가 방향성 예측이기 때문에 레이블은 (+)방향일 경우 [1, 0]으로, 0 또는 (-)방향일 경우 [0, 1]로 구분
        if (label.iloc[i] - label.iloc[i+1]) > 0:
            y_list.append([1])
        else:
            y_list.append([0])
    return pd.DataFrame(x_list), pd.DataFrame(y_list)
